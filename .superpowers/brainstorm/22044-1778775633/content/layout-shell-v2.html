Note on desktop mirror: the GitLab-style inline task-detail panel typically lives on the right edge. If desktop mirror is enabled and sidebar moves right, the detail panel flips to the left — the two never overlap. Default keeps mirror OFF on desktop so most users get the conventional layout. Power users can flip everything.
