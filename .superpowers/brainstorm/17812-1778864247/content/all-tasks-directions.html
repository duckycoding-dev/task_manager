All tasks · /tasks — 3 directions

Same data + same backend; different interaction philosophies. Sort/group/filter, saved views, bulk actions vary per option.

Simple horizontal chip bar at top. Click chip → toggle filter (multi-select OK). Auto-grouped sections by relative date (Overdue / Today / Tomorrow / This week / Later / No date). No bulk actions in v1. Inline checkbox marks complete.
