Calendar — 3 directions

Tasks (with dueDate) + reminders (with remindAt) plotted on the calendar. View switcher: Month / Week / Day. Mobile fallback shown per option.

A · Month grid with task titles in cells (Google Calendar-ish)

Each day cell shows the first 2-3 task titles + a "+N more" link. Click title → detail panel. Click day number → All-tasks page filtered to that day. Click empty cell → quick-add task for that date.
